## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
````

### 2️⃣ Backend Setup

```bash
cd backend
npm install
```

Create a **.env** file in `backend/`:

```env
MONGO_URI=mongodb://127.0.0.1:27017/yourdbname
PORT=5000
```

Run backend:

```bash
node server.js
```

### 3️⃣ Frontend Setup

```bash
cd project
npm install
npm run dev
```

Frontend will run at:

```
http://localhost:5173
```

Backend will run at:

```
http://localhost:5000
```
